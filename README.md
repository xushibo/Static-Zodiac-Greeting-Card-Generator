1. 产品概述
1.1. 产品名称
静态生肖祝福贺卡生成器 (Static Zodiac Greeting Card Generator)

1.2. 产品愿景
打造一个轻量、快速、无需后端服务器、可无限扩展的在线互动贺卡工具。用户在每年中国农历新年期间，可以方便地生成一张带有当年生肖特色和随机祝福语的精美电子贺卡，用于分享和传递节日祝福。

1.3. 目标用户
希望快速制作并分享新年电子贺卡的普通互联网用户。

希望在公司官网或活动页面中嵌入一个有趣的、应景的新年互动功能的企业。

1.4. 核心价值
零成本运营： 纯静态网页，可部署在任何免费的静态托管平台（如 GitHub Pages），无服务器费用。

极速体验： 所有计算和渲染均在用户浏览器端完成，响应速度极快，无网络延迟。

千人千面： 通过多层次的资源随机组合，每次生成的贺卡都独一无二。

极低维护成本： 独特的"前端资源化，逻辑客户端化"架构，使得年度内容更新变得异常简单，无需修改核心代码。

2. 核心功能
功能模块	功能描述
自动生肖识别	网页加载时，系统通过 JavaScript 自动获取客户端当前的公历年份，并根据内置算法精确计算出当年对应的农历生肖。
随机化贺卡生成	系统会从预设的资源库中随机抽取 背景图、当年生肖主图、装饰元素、生肖专属祝福语 和 通用祝福语，并动态组合成一张完整的贺卡。
"换一张"功能	用户点击"换一张祝福"按钮，系统会立即执行一次全新的随机组合，生成一张与之前完全不同的贺卡，为用户提供更多选择。
响应式设计	页面布局能自动适应不同尺寸的设备屏幕，无论是桌面电脑、平板还是手机，都能获得良好的视觉和操作体验。
3. 技术架构
3.1. 技术栈
前端框架: 无 (原生 HTML, CSS, JavaScript)

核心语言: HTML5, CSS3, JavaScript (ES6)

依赖库: 无

3.2. 架构原则
"前端资源化，逻辑客户端化"

资源化: 所有的图片、文案均被视为"数据资源"，集中存储在固定的文件目录和 assets.js 文件中。

客户端化: 所有的业务逻辑，包括生肖计算、随机数生成、DOM 元素渲染等，全部在用户的浏览器中完成。

4. 内容管理与年度维护指南
本产品的最大优势在于维护的便捷性。维护人员无需具备编程知识，只需遵循以下步骤即可完成年度更新。

4.1. 关键文件说明
唯一的需要修改的文件是 js/assets.js。它用简单的格式定义了所有的祝福语和图片数量。

4.2. 年度更新流程 (以2026马年为例)
第一步：准备并上传新图片

准备图片： 请设计师提供若干张（例如3张）新的"马"年主题图片。

上传图片： 将这些图片命名为 1.png, 2.png, 3.png，并上传到服务器的 assets/images/zodiac/horse/ 文件夹内。

第二步：修改 assets.js 文件

打开文件： 使用任何文本编辑器打开 js/assets.js。

更新祝福语： 找到 wishes.zodiac 部分，添加马年的专属祝福语。

JavaScript
horse: ["马到成功", "一马当先", "龙马精神"],
更新图片数量： 找到 images.zodiac 部分，声明马年主图的数量。

JavaScript
horse: 3, // 因为我们上传了3张图
保存并上传文件。

维护完成！ 当时间进入2026年，网站将自动开始展示马年主题贺卡。

4.3. CDN资源支持
为了让项目更加灵活，我们支持从CDN加载图片资源，这样可以减少服务器负载并提高加载速度。

在 js/assets.js 文件中，有一个 cdnImages 对象，可以在其中添加CDN图片链接。系统会优先使用CDN资源，如果没有配置CDN资源，则回退到本地资源。

示例配置：
```javascript
cdnImages: {
    backgrounds: [
        "https://example.com/background1.jpg",
        "https://example.com/background2.jpg"
    ],
    decorations: [
        "https://example.com/decoration1.png",
        "https://example.com/decoration2.png"
    ],
    zodiac: {
        rat: [
            "https://example.com/rat1.png",
            "https://example.com/rat2.png"
        ]
    }
}
```

4.4. 获取图片资源的推荐渠道
为了帮助您更好地完善项目，以下是推荐的一些免费商用图片资源网站：

1. Unsplash (https://unsplash.com) - 高质量免费图片
2. Pixabay (https://pixabay.com) - 免费图片、插图、矢量图
3. Pexels (https://pexels.com) - 免费图片和视频
4. 图精灵 (https://616pic.com) - 提供大量免费可商用的中国风素材
5. 千图网 (https://www.58pic.com) - 有免费和付费资源，注意筛选"可商用"标识
6. 我图网 (https://www.588ku.com) - 提供免费商用素材

获取资源的步骤：
1. 访问上述任一网站
2. 搜索关键词："中国新年背景"、"生肖动物"、"中国风装饰元素"等
3. 筛选"免费"和"可商用"选项
4. 下载JPG/PNG格式图片或将CDN链接添加到 assets.js
5. 按照项目结构放入对应文件夹或更新 js/assets.js 中的CDN配置
6. 更新 js/assets.js 中的图片数量配置

注意事项：
- 背景图片建议使用JPG格式，命名为 1.jpg, 2.jpg, 3.jpg...
- 装饰元素和生肖图片建议使用PNG格式（支持透明背景），命名为 1.png, 2.png...
- 每个生肖文件夹中的图片数量需在 assets.js 中准确配置
- 如果使用CDN资源，请确保链接长期有效
